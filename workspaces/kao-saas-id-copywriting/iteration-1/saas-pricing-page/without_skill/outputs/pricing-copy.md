## FAQ

**Apakah paket Gratis benar-benar gratis?**
Ya, paket Gratis bisa digunakan tanpa batas waktu. Tidak ada biaya tersembunyi. Anda bisa upgrade kapan saja saat bisnis berkembang.

**Apakah ada biaya tambahan untuk integrasi GoFood/GrabFood?**
Tidak. Integrasi GoFood dan GrabFood sudah termasuk di paket Pro tanpa biaya tambahan.

**Bagaimana cara upgrade dari paket Gratis ke Pro?**
Anda bisa upgrade langsung dari dashboard CloudKas. Data dan pengaturan Anda akan tetap tersimpan.

**Apakah ada kontrak jangka panjang?**
Tidak ada. Semua paket bersifat bulanan dan bisa dibatalkan kapan saja.

**Metode pembayaran apa saja yang didukung untuk langganan?**
Anda bisa membayar langganan CloudKas via transfer bank, kartu kredit/debit, atau e-wallet.

**Apakah data saya aman?**
CloudKas menggunakan enkripsi standar industri dan server yang berlokasi di Indonesia. Data Anda dibackup secara otomatis setiap hari.
